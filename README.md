## Git Cheatsheet
Compare changes with local branch
```
git diff
```
Add files to stage
```
git add <files>
```
Commit to local branch
```
git commit -m "message"
```
Push to remote branch
```
git push
```
Checkout branch
```
git checkout master
```
Git status
```
git status
```
